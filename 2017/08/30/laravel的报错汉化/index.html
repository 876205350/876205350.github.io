laravle 中 validate.php 汉化

发表于 2017-08-30 | 分类于 laravel

错误提示汉化:

位于\resources\lang\en\validation.php里面的提示全是英文

与en同级创建文件夹zh将en文件夹里内容粘贴到zh

将zh文件夹下的validation.php内容替换

最后一步

打开config/app.php 文件 将其中 ‘locale’ => ‘en’,修改为’locale’ => ‘zh’, 即可
